PathConverter

typealias PathConverter = (Path, FileSystem) -> Path (source)

Platform and version requirements: JVM

A function which converts a Path object to a Path of the given FileSystem.

This is only required to handle relative paths.

Exceptions

InvalidPathException - The given Path cannot be converted to a Path of the given FileSystem.

Return
A Path that is equivalent to the given Path and associated with the given FileSystem.
