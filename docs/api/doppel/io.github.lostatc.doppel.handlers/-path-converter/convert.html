convert

abstract fun convert(path: Path, fileSystem: FileSystem): Path (source)

Platform and version requirements: JVM

Converts path to a Path associated with fileSystem.

This is only required to handle relative paths.

Exceptions

InvalidPathException - The given Path cannot be converted to a Path of the given FileSystem.

Return
A Path that is equivalent to path but associated with fileSystem.
