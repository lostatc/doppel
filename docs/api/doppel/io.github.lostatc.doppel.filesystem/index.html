Package io.github.lostatc.doppel.filesystem

Platform and version requirements: JVM

Classes for accessing and modifying the filesystem.

This package provides the PathDelta class which allows changes to the filesystem to be queued up in memory and then applied all at once. Different types of changes that can be applied are represented by implementations of FilesystemAction.
