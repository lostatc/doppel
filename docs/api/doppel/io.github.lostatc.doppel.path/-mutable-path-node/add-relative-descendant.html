addRelativeDescendant

fun addRelativeDescendant(pathNode: MutablePathNode): Unit (source)

Platform and version requirements: JVM

Adds the given pathNode as a descendant of this node, inserting it into the tree.

The given path node is assumed to be relative to this node. If a node with the same path as the given node already exists, it is replaced.

Exceptions

IllegalArgumentException - The given path node is absolute.
