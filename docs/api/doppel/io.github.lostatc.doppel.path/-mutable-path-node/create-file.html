createFile

fun createFile(recursive: Boolean, onError: ErrorHandler): Unit (source)

Overrides PathNode.createFile

Platform and version requirements: JVM

Creates the file represented by this path node in the filesystem.

What type of file is created is determined by the type.

The following exceptions can be passed to onError:

  • IOException: Some I/O error occurred while creating the file.

Parameters

recursive - Create this file and all its descendants.

onError - A function that is called for each I/O error that occurs and determines how to handle them.
