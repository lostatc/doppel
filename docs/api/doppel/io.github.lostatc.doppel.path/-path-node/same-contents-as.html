sameContentsAs

abstract fun sameContentsAs(other: PathNode): Boolean (source)

Platform and version requirements: JVM

Returns whether the files represented by this path node and other have the same contents.

How file contents are compared is determined by the type. If the type of each node is different, this returns false.
