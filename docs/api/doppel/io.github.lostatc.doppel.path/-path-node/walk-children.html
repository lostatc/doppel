walkChildren

abstract fun walkChildren(direction: WalkDirection = WalkDirection.TOP_DOWN): Sequence<PathNode> (source)

Platform and version requirements: JVM

Returns a sequence of all the descendants of this node.

This walks through the tree of children depth-first regardless of which direction is being used. This node is not included in the sequence.

Parameters

direction - The direction in which to walk the tree.
