createFile

abstract fun createFile(recursive: Boolean = false, errorHandler: ErrorHandler = ThrowHandler()): Unit (source)

Platform and version requirements: JVM

Creates the file represented by this path node in the file system.

What type of file is created is determined by the type.

The following exceptions can be passed to errorHandler:

  • IOException: Some I/O error occurred while creating the file.

Parameters

recursive - Create this file and all its descendants.

errorHandler - A function that is called for each error that occurs and determines how to handle them.
