relativize

abstract fun relativize(other: PathNode): PathNode (source)

Platform and version requirements: JVM

Returns a copy of other which is relative to this path node.

If this path node is "/a/b" and other is "/a/b/c/d", then the resulting path node will be "c/d".

If the path represented by this node is absolute, then other must be absolute. Similarly, if the path represented by this node is relative, then other must also be relative.

Exceptions

IllegalArgumentException - other is not a path node that can be relativized against this node.
