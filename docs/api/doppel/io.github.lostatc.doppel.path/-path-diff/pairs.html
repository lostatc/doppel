pairs

val pairs: Set<PathPair> (source)

Platform and version requirements: JVM

Pairs of absolute paths of descendants. If a path exists in one node but not the other, the missing path will be null. The left path is associated with the filesystem of left, and the right path is associated with the filesystem of right.
