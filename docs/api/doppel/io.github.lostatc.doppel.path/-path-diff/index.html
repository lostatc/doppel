Properties

common

 val common: Set<Path>

The relative paths of descendants that exist in both left and right.

different

 val different: Set<Path>

The relative paths of descendants that are different in both left and right. Files are different if they have the same relative path and different contents. How file contents are compared is determined by the type.

 val left: PathNode

The first path node to compare.

leftNewer

 val leftNewer: Set<Path>

right

 val right: PathNode

The second path node to compare.

rightNewer

 val rightNewer: Set<Path>

The relative paths of descendants that were modified more recently in right than in left.

rightOnly

 val rightOnly: Set<Path>

The relative paths of descendants that exist in right but not left.

same

 val same: Set<Path>

The relative paths of descendants that are the same in both left and right. Files are the same if they have the same relative path and the same contents. How file contents are compared is determined by the type.
