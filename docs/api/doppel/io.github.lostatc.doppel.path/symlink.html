symlink

fun MutablePathNode.symlink(firstSegment: String, vararg segments: String, target: Path): MutablePathNode (source)

Platform and version requirements: JVM

Builds a path node from the given file segments with a type of SymbolicLinkType.

This is meant to be used with MutablePathNode.of to construct a tree of path nodes.

Parameters

firstSegment - The first segment of the new path node.

segments - The remaining segments of the new path node.

target - The path the link points to.
