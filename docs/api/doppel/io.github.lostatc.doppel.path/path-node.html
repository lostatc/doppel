pathNode

fun MutablePathNode.pathNode(firstSegment: String, vararg segments: String, type: FileType): MutablePathNode (source)

Platform and version requirements: JVM

Builds a path node from the given path segments and file type.

This can be used to create builder methods for use with MutablePathNode.of to create a tree of path nodes.

Parameters

firstSegment - The first segment of the new path node.

segments - The remaining segments of the new path node.

type - The initial type of the created node.
