fun MutablePathNode.unknown(firstSegment: String, vararg segments: String): MutablePathNode (source)

Platform and version requirements: JVM

Builds a path node from the given file segments with a type of UnknownType.

This is meant to be used with MutablePathNode.of to construct a tree of path nodes.

Parameters

firstSegment - The first segment of the new path node.

segments - The remaining segments of the new path node.
